# Olatua

Parte de olas de la costa de Bizkaia. Next.js 14 + Tailwind + Supabase.

Fase 1: condiciones de hoy por spot (Open-Meteo, gratis y sin clave).
Fase 2: login anonimo y calendario "quien va" (Supabase).

## Avviarlo in locale (opzionale)

Serve Node 20. Dalla cartella:

```
npm install
npm run dev
```

Poi http://localhost:3000
